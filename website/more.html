QNote was made by me Cooper Ransom (or baddeveloper) to improve upon Windows Notepad while staying as minimal and efficient as possible. I wanted something without telemetry and that isn't constantly connecting to the internet for no fucking reason. This slightly improves upon the default Windows Notepad, has more features is stable and the biggest one: open-source (oh yea and no stupid AI). Anyone is free to contribute to QNote with new additions or feedback, thank you for using QNote.
